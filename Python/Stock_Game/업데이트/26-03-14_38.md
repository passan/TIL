### 3) Frontend Fallback 및 렌더링
- **파일**: [dashboard.js](file:///d:/Antigravity/Stock_Game/static/js/dashboard.js)
- **라인**: 690-698
- **수정 내용**: 데이터가 `undefined`이거나 `null`일 경우 `0%` 또는 `사유 없음`으로 표시되도록 안전장치 마련.

```javascript
// static/js/dashboard.js (L690-698)
const rate = item.rate !== undefined && item.rate !== null ? item.rate : 0;
const reason = item.reason || "상세 사유 없음";

tr.innerHTML = `
    <td class="px-6 py-4 text-xs font-mono text-slate-400">${item.date}</td>
    <td class="px-6 py-4 font-bold ${typeColor}">${item.type}</td>
    <td class="px-6 py-4 font-extrabold text-slate-800">${item.name}</td>
    <td class="px-6 py-4 font-mono text-slate-700">${item.price.toLocaleString()}원</td>
    <td class="px-6 py-4 font-bold ${rate >= 0 ? 'text-red-500' : 'text-blue-500'}">${rate >= 0 ? '+' : ''}${rate}%</td>
    <td class="px-6 py-4 text-xs text-slate-500 leading-relaxed max-w-xs">${reason}</td>
`;
```
